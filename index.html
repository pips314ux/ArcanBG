Typography

Our typography system is built on Poppins, a geometric sans-serif typeface that reflects our brand's innovative and forward-thinking nature. The type system is designed for optimal readability and visual hierarchy across all digital platforms.

Typeface

Poppins

Primary Typeface
Aa

Poppins is a geometric sans-serif typeface with a modern and friendly feel. Its clean circles and curves give it a distinct personality, making it suitable for headings and display text.

Available weights: 400, 500, 600, 700

San Francisco

Secondary Typeface / System Font
Aa

San Francisco is an Apple designed typeface that provides a consistent, legible, and friendly typographic voice. It's optimized for display text at various sizes and in a wide range of languages across multiple interfaces. SF Pro, a variant of San Francisco, is the system font for iOS, iPadOS, macOS, and tvOS.

System font on Apple platforms Optimized for legibility Used for body text and UI elements

Usage Guidelines

Logo Usage

✓

Do's

  • Maintain clear space equal to the height of the "A" in the logo
  • Use approved color variations for different backgrounds
  • Keep the logo proportional and undistorted
  • Use high-resolution versions for print and digital
  • Place the logo on solid or gradient backgrounds
✕

Don'ts

  • Don't distort, stretch, or modify the logo
  • Don't change the approved brand colors
  • Don't add effects, shadows, or outlines
  • Don't use low-resolution versions
  • Don't place the logo on busy backgrounds

Color Usage

✓

Color Principles

  • Use AI Blue for primary actions and brand elements
  • Apply Web3 Green for success states and Web3 features
  • Maintain proper contrast ratios for accessibility
  • Use the neutral palette for UI elements and backgrounds
  • Apply accent colors sparingly for emphasis
✕

Color Restrictions

  • Don't create new color combinations
  • Don't use colors outside the approved palette
  • Don't apply colors that reduce readability
  • Don't use gradient variations not in guidelines
  • Don't mix brand colors with arbitrary colors

Typography Usage

✓

Type Principles

  • Use Poppins as the primary typeface
  • Follow the established type scale
  • Maintain proper line heights and spacing
  • Use appropriate weights for hierarchy
  • Ensure text remains legible at all sizes
✕

Type Restrictions

  • Don't use alternative typefaces
  • Don't modify letter spacing or line height
  • Don't use custom font weights
  • Don't stretch or distort text
  • Don't use decorative text effects

Digital Applications

✓

Best Practices

  • Use SVG format for logos when possible
  • Maintain responsive design principles
  • Follow accessibility guidelines
  • Use appropriate file formats for each context
  • Optimize assets for web performance
✕

Avoid These

  • Don't use raster formats for logos
  • Don't ignore responsive breakpoints
  • Don't compromise accessibility
  • Don't use unoptimized assets
  • Don't mix different style versions
